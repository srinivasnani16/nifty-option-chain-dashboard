# Nifty Option Chain Live Dashboard

A real-time Nifty 50 Option Chain dashboard built on AWS EC2.

## Features
- Live NSE options data via Angel One SmartAPI (every 5 minutes)
- Dark-themed Flask web dashboard with password protection
- TradingView live chart embed
- SQLite database storage
- Auto-restarts on EC2 reboot via crontab

## Tech Stack
Python, Flask, SQLite, Angel One SmartAPI, AWS EC2
